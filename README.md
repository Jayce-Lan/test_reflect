# 反射
> 将类的各个组成部分封装为其它对象，这就是反射机制

## 反射的好处
- 可以在程序的运行过程中操作对象
- 可以解耦，提高程序的可扩展性

## 获取Class类的对象
- `Class.forName("全类名");`	 将字节码文件加载进内存，返回Class对象（该方式适合文件未加载进内存时使用）
- `类名.class` 通过类名.class获取Class类的对象（该方式适用于文件已经加载进入内存）
- `对象.getClass();` getClass()方法封装在了Object对象中（该方式适用于对象已经被创建，也就是被new出来后）
